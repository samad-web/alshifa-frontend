
# Al Shifa Aayush Healthcare Platform

This project is a custom-built healthcare platform, designed and engineered by Sirah Digital.

## Overview

Al Shifa Aayush is a modern, full-stack healthcare management system. It provides secure authentication, role-based dashboards, patient-doctor assignment, and robust admin features for clinics and hospitals.

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS (customized)
- Node.js + Express backend
- Prisma ORM + PostgreSQL

## Getting Started

1. **Clone the repository:**
	```sh
	git clone <YOUR_GIT_URL>
	cd <YOUR_PROJECT_NAME>
	```
2. **Install dependencies:**
	```sh
	npm install
	```
3. **Start the development server:**
	```sh
	npm run dev
	```

## Project Structure

- `src/` — Frontend source (components, pages, hooks, services)
- `backend/` — Node.js/Express backend (routes, services, prisma)
- `public/` — Static assets
